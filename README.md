# PHP Mini Projects

A collection of **basic to intermediate PHP mini projects** created for learning and practicing core PHP concepts such as arrays, conditionals, forms, and simple UI styling.

This repository is ideal for:

* Beginners learning PHP
* Academic / college assignments
* Quick revision of PHP fundamentals
* Mini-projects for portfolios

---

##  Projects Included

### 1. Calculator (`calculator.php`)

A simple PHP-based calculator that performs basic arithmetic operations.

**Concepts used:**

* HTML forms
* PHP conditionals
* GET/POST handling

---

### 2. Day of Week Finder (`dayofweek.php`)

Finds the day of the week based on user input.

**Concepts used:**

* Switch-case
* User input handling
* PHP logic

---

### 3. Info Card (`infocard.php`)

Displays user information in a styled card layout.

**Concepts used:**

* PHP variables
* HTML + CSS styling
* Data display

---

### 4. Product List (`productlist.php`)

Displays a simple list of products using PHP arrays.

**Concepts used:**

* Indexed & associative arrays
* Loops (`foreach`)

---

### 5. Product Catalog (`productcatalog.php`)

A **mini product catalog system** with filtering and sorting.

**Features:**

* Filter by category
* Sort by name, price, or rating
* Minimum price filter
* Clean, professional UI

**Concepts used:**

* Multidimensional arrays
* `array_filter()` and `usort()`
* GET parameters
* Secure output using `htmlspecialchars()`

---

##  Technologies Used

* PHP (Core PHP)
* HTML5
* CSS3

No frameworks or databases are used.

---

##  How to Run the Projects

1. Install **XAMPP** / **WAMP** / **MAMP**
2. Clone or download this repository
3. Move the folder to:

   ```
   htdocs (XAMPP)
   ```
4. Start Apache
5. Open your browser and visit:

   ```
   http://localhost/PHP-Mini-Projects/
   ```
